# Pay-per-Use Electricity Smart Meter
A decentralized electricity metering system built on Stacks blockchain.

## 🎯 Features

- User meter initialization
- STX token-based payments
- Real-time consumption tracking
- Secure rate management
- Usage history tracking

## 🔒 Security Features

- Owner-only administrative functions
- Input validation
- Balance checks
- Active meter status verification

## 🚀 Getting Started

1. Install Clarinet
```bash
curl -L https://github.com/hirosystems/clarinet/releases/download/v1.0.0/clarinet-linux-x64.tar.gz | tar xz
```

2. Initialize project
```bash
clarinet new pay-per-use-meter && cd pay-per-use-meter
```

3. Run tests
```bash
clarinet test
```
